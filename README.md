# AICW_Project_DL_AMC_Spectrograms
# Deep Learning-Assisted Automatic Modulation Classification

# Project Overview
This project focuses on Automatic Modulation Classification (AMC) using Deep Learning techniques. It uses Convolutional Neural Networks (CNNs) trained on spectrograms to classify modulation schemes like BPSK and QPSK in wireless communication systems.

# Problem Statement
Traditional modulation classification techniques fail in noisy and dynamic environments. There is a need for a robust and intelligent system that can accurately classify modulation types under varying conditions.

# Objectives
- To understand digital modulation techniques  
- To design an AMC system using Deep Learning  
- To eliminate manual feature extraction  
- To improve classification accuracy under different SNR levels  

# Proposed Methodology
- RF signals are converted to baseband  
- Signals are preprocessed and normalized  
- Spectrograms are generated  
- CNN model is trained on spectrogram data  
- Model classifies modulation (BPSK, QPSK)  
- Output signal is demodulated  

# Technologies Used
- Programming Language: Python  
- Libraries: TensorFlow, Keras, NumPy, SciPy  
- Tools: Jupyter Notebook, VS Code  

# System Requirements
- Minimum 8GB RAM  
- GPU (optional for faster training)  
- OS: Windows / Linux  

# Applications
- V2X Communication  
- Cognitive Radio Networks  
- 5G / 6G Systems  
- Internet of Things (IoT)  
- Spectrum Monitoring  

# Expected Outcomes
- Accurate modulation classification system  
- Better performance in noisy environments  
- Reduced manual effort using deep learning  

# Conclusion
The proposed CNN-based model improves classification accuracy and effectively distinguishes between similar modulation schemes like BPSK and QPSK.
